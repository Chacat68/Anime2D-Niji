AI画图服务商配置指南

🤖 OpenAI

API 地址： https://api.openai.com/v1
支持模型：
  • dall-e-3 - 最新版本，高质量图像生成
  • dall-e-2 - 经典版本，速度较快
获取 API Key：
  1. 访问 OpenAI 官网注册
  2. 进入 API Keys 页面
  3. 点击 "Create new secret key" 创建密钥
官方文档： https://platform.openai.com/docs/guides/images
定价：
  • DALL-E 3: 约 $0.04 - $0.12 每张图
  • DALL-E 2: 约 $0.016 - $0.020 每张图

🌟 Google Gemini

API 地址： https://generativelanguage.googleapis.com/v1beta
支持模型：
  • gemini-2.5-flash-image (Nano Banana) - 快速生成
  • gemini-3-pro-image-preview (Nano Banana Pro) - 4K高级图像
  • gemini-2.0-flash-exp - 实验性版本
获取 API Key：
  1. 访问 Google AI Studio
  2. 使用 Google 账号登录
  3. 点击 "Get API Key" 或 "Create API Key"
官方文档： https://ai.google.dev/gemini-api/docs/image-generation
免费额度：
  • 每分钟 15 次请求
  • 每天 1500 次请求
  • 完全免费（截至2025年）
注意事项：

需要在请求 URL 中添加 API Key：
?key=YOUR_API_KEY

☁️ Azure OpenAI

API 地址格式： https://YOUR_RESOURCE_NAME.openai.azure.com
支持模型：
  • dalle3 - 需要在 Azure 部署
获取访问权限：
  1. 访问 Azure 门户
  2. 创建 "Azure OpenAI" 资源
  3. 在资源中部署 DALL-E 3 模型
  4. 从 "密钥和终结点" 获取 API Key 和终结点地址
官方文档： https://learn.microsoft.com/azure/ai-services/openai/dall-e-quickstart
配置说明：
  • 需要单独的 Azure 订阅
  • API Key 在 Azure 门户中获取
  • 终结点地址因资源而异

🏠 本地 API

默认地址： http://localhost:8080/v1
适用场景：
  • 本地部署的 Stable Diffusion WebUI
  • 支持 OpenAI 格式的本地服务
  • 自建 AI 画图服务
推荐工具：
  • Stable Diffusion WebUI - 需要配置 API 扩展
  • ComfyUI - 支持自定义工作流
  • Web Stable Diffusion - 浏览器内运行
配置提示：

根据本地服务的实际端口和路径修改 API 地址，一般不需要 API Key。

🔧 自定义服务

说明：

适用于任何兼容 OpenAI 图像生成 API 格式的第三方服务。

常见第三方服务：
  • Cloudflare Workers AI
  • Replicate
  • Hugging Face Inference API
  • 各种 OpenAI 兼容的代理服务
配置方法：
  1. 填写服务商提供的 API 终结点地址
  2. 填写相应的 API Key
  3. 填写支持的模型名称
  4. 测试连接确认可用

💡 使用提示

API Key 安全

  • 不要分享你的 API Key
  • 定期轮换密钥
  • 监控 API 使用量

提示词建议

  • 使用英文提示词效果更好
  • 详细描述场景、风格、光照等
  • 避免模糊或矛盾的描述

费用控制

  • Google Gemini 提供免费额度
  • OpenAI 按张数计费
  • 本地运行无 API 费用

故障排查

  • 检查 API Key 是否正确
  • 确认网络连接正常
  • 查看控制台错误信息（F12）
